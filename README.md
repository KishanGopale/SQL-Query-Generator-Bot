# 🧑‍💻 SQL Query Generator using Google Gemini

This is a **mini chatbot** built with **Streamlit** and **Google Gemini API** that converts simple English text into **SQL queries**.  
It also provides:
- ✅ The generated SQL query  
- ✅ A sample **expected output table**  
- ✅ A **simple explanation** of the query  

Perfect for beginners learning SQL or anyone who wants quick query generation from natural language. 🚀  

---

## ⚙️ Features
- Convert **Plain English → SQL Query**
- Generate **Expected Output** for the query
- Provide **Step-by-step Explanation**
- Simple **Streamlit UI** 

The app will generate:  
- SQL query  
- Expected tabular output  
- Explanation of how the query works
